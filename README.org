* RustTalk
#+CAPTION: RustTalk logo

RustTalk 是一档专注在 Rust 社区的程序员闲聊节目。节目主旨：
- 专注 Rust 社区，探索 Rust 社区背后的 *奇人轶事*
- 不追求「枯燥无味」的干货，更关注「耐人寻味」经验分享
