为什么办 RustTalk

  • Rust 语言中文社区是一个偏干货分享的地方，偏文字，RustTalk 更侧重“湿货”，不仅仅会介绍到 Rust 的设计理念，更多的会去挖掘 Rust 背后的 奇人轶事

  • 学习一门技术最好的方式是实践，播客也算一种

  • 王垠 2016 的文章：对 Rust 语言的分析，PDF 备份
